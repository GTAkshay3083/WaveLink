# WaveLink – Product Requirements Document (PRD)

## Project Name
WaveLink

## Tagline
Turn every phone into part of the speaker.

## Vision
WaveLink transforms multiple smartphones into one synchronized speaker system using a local Wi-Fi network.

Instead of relying on a Bluetooth speaker, one phone becomes the host and streams audio to nearby phones.

## Problem

Many situations don't have a speaker available:

- College hostels
- Birthday parties
- Trips
- Camping
- Small gatherings

Everyone has a phone, but only one phone's speaker is used.

WaveLink allows everyone to contribute their phone's speaker.

## Target Users

- College students
- Hostel residents
- Friends at parties
- Travellers
- Anyone without a Bluetooth speaker

## Version 1 Goals

### Host

- Create a room
- Give the room a custom name
- Browse local music library
- Search songs
- Select a song
- Stream audio to connected devices
- Control playback (Play, Pause, Stop, Next)
- View connected devices

### Guest

- Join via QR Code
- Join via Room Code
- Receive synchronized audio
- Control only local device volume
- Leave room

### Connectivity

- Same Wi-Fi support
- Mobile Hotspot support
